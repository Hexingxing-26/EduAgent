# 新手指南 - 如何启动AI智能体模块

## 📋 启动前准备

- [ ] 已安装 Python 3.10 或更高版本
- [ ] 已获取项目代码（从Git克隆或直接拷贝）
- [ ] 已获取讯飞API密钥（向负责人索取）

---

## 🚀 快速启动（3步搞定）

### 第1步：进入项目目录

将终端切换到 `AI_Agent_System` 文件夹：
```bash
cd AI_Agent_System
```

### 第2步：配置API密钥

1. 复制 `.env.example` 并重命名为 `.env`
2. 用记事本打开 `.env` 文件
3. 填入讯飞API密钥：
   ```
   SPARK_API_KEY="你的API_KEY"
   SPARK_API_SECRET="你的API_SECRET"
   ```

### 第3步：启动服务器

**Windows 用户**：双击 `启动服务器.bat`

**其他系统**：在终端执行：
```bash
pip install -r requirements.txt
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

## ❓ 常见问题

### Q1: 提示 "ModuleNotFoundError: No module named 'xxx'"
**A**: 依赖没装好，执行：
```bash
pip install -r requirements.txt
```

### Q2: 提示 "API 错误 (HTTP 401)"
**A**: API密钥配置错误，请检查 `.env` 文件中的密钥是否正确。

### Q3: 端口8000被占用怎么办？
**A**: 修改启动命令的端口号：
```bash
python -m uvicorn app.main:app --host 0.0.0.0 --port 8080 --reload
```
然后访问 http://localhost:8080/

### Q4: 生成的内容是"生成失败，请稍后重试"
**A**: LLM调用失败，可能原因：
- API密钥错误
- 网络不通
- API配额用完

没有API密钥也能正常运行，系统会使用备用内容兜底。

### Q5: 如何停止服务器？
**A**: 在终端按 `Ctrl + C`
