=======================================================================
                      实验1：Python环境搭建与基础编程
=======================================================================

【实验目的】
1. 安装Python开发环境
2. 掌握交互式解释器的使用
3. 掌握Python程序编写与运行的基本流程
4. 掌握变量、表达式、基本输入输出

【实验内容】

1.1 环境准备（20分钟）
- 从python.org下载并安装Python
- 验证安装：在命令行输入 python --version
- 熟悉IDLE或VS Code的基本操作

1.2 交互式解释器练习（20分钟）
在交互式解释器中完成以下操作，记录输出结果：
1. 计算 123 + 456 的结果
2. 计算 2 ** 10 的结果
3. 计算 17 // 3 和 17 % 3 的结果
4. 计算 (3 + 5) * 2 的结果

1.3 变量与数据类型练习（20分钟）
编写简单的表达式并观察结果：
1. 创建变量 x = 10，y = 20，计算 x + y
2. 创建字符串变量 name = "Python"，输出 "Hello, Python"
3. 将用户输入的字符串转换为整数，并计算其平方

1.4 编写第一个程序（30分钟）
创建文件 hello.py，包含以下功能：
1. 打印 "欢迎来到Python世界！"
2. 使用 input() 获取用户姓名，输出 "你好，{姓名}！"
3. 获取用户输入的两个数字，计算并输出它们的和
