一、选择题（每题4分，共40分）

1. 以下哪个关键字用于定义类？
   A. class    B. def    C. struct    D. object

2. 类中 __init__ 方法是？
   A. 析构函数    B. 构造函数    C. 普通方法    D. 静态方法

3. 方法中的 self 参数代表什么？
   A. 类本身    B. 当前实例    C. 父类    D. 一个普通参数

4. 以下哪个是继承的语法？
   A. class Sub(Super):    B. class Sub extends Super:
   C. class Sub : Super    D. 以上都不对

5. 多态是指？
   A. 同一个方法在不同对象中有不同行为    B. 隐藏对象内部细节
   C. 代码重用    D. 以上都是

6. 以下哪个用于调用父类构造函数？
   A. super().__init__()    B. parent.__init__()
   C. self.__parent__()    D. 以上都可以

7. 以下哪个是正确的方法重写方式？
   A. 在子类中定义同名方法    B. 使用 @override 装饰器
   C. 使用 override 关键字    D. 以上都不对

8. 以下哪个用于检查对象是否是类的实例？
   A. isinstance()    B. issubclass()    C. type()    D. hasattr()

9. 多重继承是指？
   A. 一个类继承多个父类    B. 一个类被多个子类继承
   C. 多层继承    D. 以上都不对

10. @abstractmethod 装饰器用于？
    A. 定义抽象方法    B. 定义静态方法
    C. 定义类方法    D. 定义特性


二、判断题（每题3分，共15分）

1. Python 支持多重继承。（ ）
2. 类的属性可以在外部直接访问。（ ）
3. 私有属性用 __ 开头，无法被访问。（ ）
4. 所有类都必须继承自 object。（ ）
5. 类方法不需要 self 参数。（ ）


三、填空题（每题3分，共15分）

1. 类的构造函数是 ________ 方法。
2. 继承时使用 ________ 关键字调用父类方法。
3. 检查类是否是另一个类的子类，使用 ________ 函数。
4. 以 ________ 开头的属性名会被执行名称修饰。
5. 抽象基类需要继承自 ________。


四、编程题（每题10分，共30分）

1. 定义一个 Person 类，包含 name 和 age 属性，以及 greet() 方法。

2. 从 Person 派生 Student 类，添加 grade 属性和 study() 方法。

3. 定义一个 Rectangle 类，包含 width 和 height，实现面积计算。
