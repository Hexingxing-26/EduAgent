一、选择题（每题4分，共40分）

1. 以下哪个用于捕获异常？
   A. try/except    B. try/catch    C. try/finally    D. 以上都是

2. 以下哪个是 Python 内置异常？
   A. IndexError    B. KeyError    C. ValueError    D. 以上都是

3. 哪个关键字用于主动引发异常？
   A. throw    B. raise    C. except    D. finally

4. 以下哪个是自定义异常的正确方式？
   A. class MyError(Exception):    B. class MyError:
   C. class MyError(Error):    D. 以上都不对

5. finally 子句的特点是？
   A. 仅在发生异常时执行    B. 仅在未发生异常时执行
   C. 无论是否发生异常都执行    D. 只在捕获异常后执行

6. else 子句在 try/except 中的作用是？
   A. 发生异常时执行    B. 未发生异常时执行
   C. 总是执行    D. 用于处理所有异常

7. 以下哪个是捕获所有异常的正确写法？
   A. except:    B. except Exception:    C. 以上都是    D. 以上都不是

8. 以下哪个模块用于发出警告？
   A. warnings    B. logging    C. sys    D. os

9. 异常的传播方向是？
   A. 从主程序向函数传播    B. 从函数向调用者传播
   C. 不传播    D. 随机传播

10. 以下哪个不是 Python 标准异常？
    A. ZeroDivisionError    B. FileNotFoundError
    C. IOException    D. TypeError


二、判断题（每题3分，共15分）

1. 异常可以使用 raise 语句手动触发。（ ）
2. finally 子句可以省略。（ ）
3. except: 捕获所有异常是推荐的做法。（ ）
4. 自定义异常必须继承自 Exception 类。（ ）
5. 异常处理中只能有一个 except 子句。（ ）


三、填空题（每题3分，共15分）

1. 捕获异常使用 ________ 语句。
2. 异常的基类是 ________。
3. 使用 ________ 关键字可以重新抛出异常。
4. 警告的模块是 ________。
5. 异常处理中，________ 子句在异常发生时执行。


四、编程题（每题10分，共30分）

1. 编写程序，处理用户输入数字时的 ValueError 异常。

2. 定义一个自定义异常 NegativeValueError，在函数中检查参数为负数时引发。

3. 编写一个安全的除法函数，处理除零异常，并在 finally 中打印 "计算完成"。
