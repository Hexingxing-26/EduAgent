一、选择题（每题4分，共40分）

1. 以下哪个是Python中的条件语句？
   A. if    B. for    C. while    D. switch

2. 以下哪个值在布尔上下文中被视为 False？
   A. 0    B. 1    C. "False"    D. [0]

3. for i in range(3) 会迭代几次？
   A. 2次    B. 3次    C. 4次    D. 1次

4. 以下哪个语句用于跳出循环？
   A. continue    B. break    C. return    D. exit

5. 列表推导式 [x for x in range(10) if x % 2 == 0] 的结果是？
   A. [0, 2, 4, 6, 8]    B. [1, 3, 5, 7, 9]
   C. [0, 1, 2, 3, 4]    D. [2, 4, 6, 8]

6. 以下哪个是短路逻辑运算符？
   A. &    B. |    C. and    D. ^

7. while True: 循环的退出方式是？
   A. 自动退出    B. 使用 break    C. 使用 continue    D. 无法退出

8. 以下哪个语句用于什么都不做？
   A. pass    B. continue    C. break    D. return

9. 对于 x = [1, 2, 3]，执行 x, y, z = x 后，x, y, z 的值是？
   A. 1, 2, 3    B. [1, 2, 3], [1, 2, 3], [1, 2, 3]
   C. 报错    D. 1, 1, 1

10. 以下哪个是链式赋值的正确写法？
    A. a = b = 10    B. a = 10; b = 10    C. a, b = 10, 10    D. 以上都是

1. 使用 ________ 语句可以跳过本次循环的剩余代码。
