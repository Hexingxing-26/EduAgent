一、选择题（每题4分，共40分）

1. Python中，以下哪个是合法的变量名？
   A. 2var    B. var-2    C. _var2    D. var 2

2. 以下哪个表达式的值为4？
   A. 8 // 2    B. 8 / 2    C. 8 % 2    D. 8 ** 2

3. 以下哪个语句可以正确导入math模块并使用sqrt函数？
   A. import math; math.sqrt(9)
   B. import sqrt from math; sqrt(9)
   C. from math import sqrt; sqrt(9)
   D. 以上都可以

4. 字符串 r'C:\nowhere' 中，反斜杠的作用是？
   A. 转义字符    B. 路径分隔符    C. 原始字符串标识    D. 以上都不是

5. 以下哪个是Python中的合法字符串？
   A. "Hello"    B. 'Hello'    C. """Hello"""    D. 以上都是

6. input() 函数返回的数据类型是？
   A. int    B. float    C. str    D. list

7. 以下哪个模块用于处理复数？
   A. math    B. cmath    C. random    D. os

8. 在Python中，注释以哪个符号开头？
   A. //    B. /*    C. #    D. --

9. 以下哪个函数可以将字符串转换为整数？
   A. str()    B. int()    C. float()    D. chr()

10. Python交互式解释器的提示符是？
    A. $    B. C:\>    C. >>>    D. >>>

三、填空题（每题3分，共15分）

1. Python中使用 ________ 函数获取用户输入。
2. 打印"Hello, World!"的语句是 ________。

4. 字符串"Hello"的长度为 ________。
5. 使用 ________ 符号表示单行注释。


四、编程题（每题10分，共30分）

1. 编写一个程序，接收用户输入的两个数字，计算并输出它们的和、差、积、商。

2. 编写一个程序，让用户输入姓名和年龄，然后输出 "你好，{姓名}，你今年{年龄}岁了"。

3. 编写一个程序，计算半径为5的圆的面积（使用math模块中的pi常量）。
