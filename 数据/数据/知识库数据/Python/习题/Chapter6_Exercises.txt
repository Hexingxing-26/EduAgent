一、选择题（每题4分，共40分）

1. 以下哪个关键字用于定义函数？
   A. function    B. def    C. define    D. func

2. 函数中 return 语句的作用是？
   A. 返回函数结果    B. 终止函数执行    C. 返回 None    D. 以上都是

3. 以下哪个是函数参数的默认值写法？
   A. def func(x=10):    B. def func(=x 10):
   C. def func(x default 10):    D. 以上都不对

4. 函数中 *args 的作用是？
   A. 收集位置参数为元组    B. 收集关键字参数为字典
   C. 分配参数    D. 定义默认参数

5. 以下哪个是正确的函数调用方式？
   A. func(x=1, 2)    B. func(1, y=2)    C. func(x=1, y=2)    D. B和C都正确

6. 局部变量与全局变量同名时，哪个优先？
   A. 全局变量    B. 局部变量    C. 报错    D. 不确定

7. 递归函数必须包含什么？
   A. 循环    B. 基线条件    C. 全局变量    D. 异常处理

8. 以下哪个不是内置函数？
   A. print()    B. len()    C. factorial()    D. range()

9. 函数文档字符串通过哪个属性访问？
   A. __doc__    B. __str__    C. __repr__    D. __name__

10. 以下哪个是闭包的典型特征？
    A. 函数内部定义函数    B. 使用 global 关键字
    C. 使用 nonlocal 关键字    D. A和C都正确

4. 函数 ________ 是函数的文档字符串。
